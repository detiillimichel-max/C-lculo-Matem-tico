# Jogo de Cálculo Matemático PWA

Este é um jogo de matemática progressivo desenvolvido com HTML5, CSS3 e JavaScript. O aplicativo foi projetado como um PWA (Progressive Web App), o que significa que ele pode ser instalado no celular ou computador e funciona totalmente offline.

## Como Executar
1. Coloque todos os arquivos na mesma pasta.
2. Adicione uma imagem quadrada de sua preferência com o nome `icon.png` (tamanho recomendado: 512x512 pixels).
3. Para testar as funcionalidades de PWA (Service Worker e Instalação), abra a pasta usando um servidor local (como a extensão Live Server do VS Code) ou hospede em plataformas gratuitas como GitHub Pages ou Netlify.

## Estrutura de Fases
- **Fases 1 a 4:** Operações básicas lineares (Soma, Subtração, Multiplicação e Divisão).
- **Fases 5 a 9:** Expressões avançadas contendo Parênteses `()`, Colchetes `[]` e Chaves `{}` com dificuldades crescentes.
